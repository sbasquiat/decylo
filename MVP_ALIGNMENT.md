# Decylo MVP Alignment

## Core Purpose
**Decylo is a daily decision-making workspace** that helps users:
- Structure their thinking
- Commit to choices
- Track real outcomes
- Improve decision quality over time

**The Loop**: Think → Choose → Act → Review → Improve

---

## MVP Features Status

### ✅ 1. Today Screen
**Status**: Complete
- Daily focus input
- Streak tracking
- Add decision CTA
- Outcomes due card
- Basic stats (decisions this week, outcomes logged)

### ✅ 2. Structured New Decision Flow
**Status**: Complete
- Title + Category
- Context + Constraints
- Options with impact/effort/risk scoring
- Recommendation (suggested option)
- Final choice selection
- Confidence level
- Next action + due date

### ⚠️ 3. Decision Lifecycle
**Status**: Partially Complete
- ✅ Status flow: open → decided → completed
- ✅ Detail view
- ✅ Timeline (grouped by date)
- ❌ **Missing**: Search/filter functionality

### ✅ 4. Outcome Engine
**Status**: Complete
- ✅ Database schema supports win/neutral/loss
- ✅ Outcome display in decision detail
- ✅ Outcomes due tracking
- ✅ Outcome logging form (status, what happened, lesson)

### ✅ 5. Insights
**Status**: Complete
- Weekly decision counts
- Outcome ratio (won/neutral/lost)
- Streak tracking
- Category distribution

### ✅ 6. Data Export & Privacy
**Status**: Complete
- CSV export in Settings
- Privacy-first UX (no data selling, no ads)
- User-owned data
